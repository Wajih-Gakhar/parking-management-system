# 🅿️ Parking Management System

A vehicle parking management system built in **C++** with **Qt**, supporting multiple vehicle types with distinct pricing tiers and real-time slot tracking.

## 🚀 Features

### 🚗 Multi-Vehicle Type Support
- Supports different vehicle categories (Car, Bike, Truck, etc.)
- Each vehicle type has its own distinct pricing tier

### 🅿️ Real-Time Slot Tracking
- Live monitoring of available and occupied parking slots
- Instant slot assignment on vehicle entry

### 💰 Pricing Management
- Different pricing tiers per vehicle type
- Automatic fare calculation on exit

### 💾 Full Data Persistence
- Save, load, and delete data across sessions via file handling

## ⚙️ How to Run

### Prerequisites
- Qt Creator (6.x or later)
- C++ compiler (MinGW 64-bit)

### Steps
1. Clone the repository
```bash
   git clone https://github.com/Wajih-Gakhar/parking-management-system.git
```
2. Open `CMakeLists.txt` in **Qt Creator**
3. Select the **Desktop Qt 6.9.0 MinGW 64-bit** kit
4. Build & Run ▶️
